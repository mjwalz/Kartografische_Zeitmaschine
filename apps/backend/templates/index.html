PostGIS Database

The application is connected to a PostGIS database where all spatial data is stored. The database is optimized for geographic queries and spatial operations.

  • ✓ Full spatial indexing
  • ✓ Support for all geometry types
  • ✓ Advanced spatial queries
QGIS Integration

Connect QGIS to the PostGIS database to visualize and edit spatial data:

  1. Open QGIS
  2. Add a new PostGIS connection
  3. Use your database credentials
  4. Start working with your spatial data
